### 🧬 NAFLD Detection from Sonography — Multimodal Deep Learning
*B.Sc. Thesis · 2025–2026 (Ongoing)*
A multimodal deep learning system for automated, non-invasive detection and **5-stage staging** of Non-Alcoholic Fatty Liver Disease (Normal, Steatosis, NASH, Fibrosis, Cirrhosis) from abdominal ultrasound images fused with clinical metadata (age, BMI, ALT, AST). Designed as a scalable, cost-effective screening alternative to invasive liver biopsy for resource-limited clinical settings.
`Python` `TensorFlow` `Keras` `Attention U-Net` `Multimodal Learning` `OpenCV` `CLAHE` `SMOTE`

*Research Paper (IEEE, Under Submission) · 2025–2026 (Ongoing)*

Supports both multimodal input (image + clinical reports) and image-only input through a dynamic routing pipeline. Owned the full pipeline — data preprocessing, model architecture design, training, evaluation, and IEEE submission prep.
`Python` `PyTorch` `ResNet50` `BiLSTM` `Swin Transformer` `Attention U-Net` `Transfer Learning`
